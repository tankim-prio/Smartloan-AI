# Demo Video Script

## Intro
This is SmartLoan AI, a full-stack AI-powered loan application and review management system.

## Demo Flow
1. Login to the system.
2. Open Dashboard and explain overview cards.
3. Go to Apply page.
4. Fill applicant information.
5. Upload salary certificate, NID/passport, and photo.
6. Create loan PDF.
7. Upload PDF in Apply for Loan section.
8. Extract readable PDF text.
9. Extract structured fields.
10. Run ML prediction.
11. Send application to Review.
12. Open Review page and approve/reject application.
13. Open Reports page.
14. Open ML Model page.
15. Open AI Pilot page.
16. Show Docker running.

## Closing
SmartLoan AI demonstrates backend API development, frontend workflow, PDF automation, ML prediction, AI document assistance, Docker, and portfolio-ready system design.
