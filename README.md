# SmartLoan AI

SmartLoan AI is a full-stack AI-powered loan application and review management platform.

## Features

- Customer registration and login
- Role-based admin workflow
- Loan application form
- PDF generation
- PDF upload and text extraction
- Review approval workflow
- ML loan prediction
- Reports dashboard
- AI Pilot / RAG document assistant
- Dockerized backend and frontend

## Tech Stack

- FastAPI
- SQLite
- React + Vite
- TypeScript
- Docker
- Machine Learning with scikit-learn
- PDF extraction with PyPDF / PyMuPDF

## Run with Docker

`powershell
docker compose up --build
` 

Frontend: http://localhost:5173

Backend: http://localhost:20000
